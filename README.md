## ğŸ§¾ Glottal Stop Representation

In Tetum, the **glottal stop** is a phoneme and part of the official alphabet.  
For technical compatibility with Microsoft Word, it is represented using the **ASCII apostrophe `'` (U+0027)**.  

Examples:

la'o
ti'an
ko'alia


This is a **technical choice**, not a linguistic compromise, and ensures reliable spellchecking.
